# 💧 喝水打卡 PWA 版本

> 像真正的 APP 一样使用，无需安装 APK！

## 🎯 这是什么？

这是一个 **PWA（渐进式 Web 应用）** 版本的喝水打卡应用：
- ✅ 可以添加到手机主屏幕
- ✅ 像原生 APP 一样启动和使用
- ✅ 支持离线使用
- ✅ 所有功能完全一样
- ✅ 不需要下载安装包

## 🚀 3 步开始使用

### 方法一：在线使用（推荐）

1. **上传到 Netlify**（最简单）
   - 访问 https://app.netlify.com/drop
   - 把 `pwa` 整个文件夹拖拽到网页上
   - 等待 10 秒，自动部署完成
   - 获得一个网址（如：`https://random-name-123.netlify.app`）

2. **手机访问**
   - 用手机浏览器打开这个网址
   - 会自动弹出"安装"提示

3. **安装到主屏幕**
   - 点击"安装"按钮
   - 或使用浏览器的"添加到主屏幕"功能
   - 完成！桌面上出现 APP 图标

### 方法二：本地测试

1. **启动服务器**
   ```bash
   cd pwa
   ./启动.sh
   ```

2. **浏览器访问**
   打开 http://localhost:8080

3. **手机访问**
   确保手机和电脑在同一WiFi，访问 http://你的电脑IP:8080

## ✨ PWA vs APK

| 特性 | PWA ✅ | APK |
|------|--------|-----|
| 安装 | 浏览器直接添加 | 需下载安装包 |
| 大小 | 极小（几KB） | 较大（几MB） |
| 更新 | 自动更新 | 手动下载 |
| 跨平台 | Android + iOS | 仅Android |
| 离线 | ✅ 支持 | ✅ 支持 |

## 📱 手机上安装步骤

### Android（Chrome）
1. 打开网址
2. 点击"安装"横幅
3. 或：菜单 ⋮ → "添加到主屏幕"

### iPhone（Safari）
1. 打开网址
2. 点击分享按钮 📤
3. "添加到主屏幕"

## 🌐 在线托管服务推荐

### 🥇 Netlify（最推荐）
- 网址: https://app.netlify.com/drop
- 优点: 拖拽上传，自动部署，免费 HTTPS
- 操作: 把 pwa 文件夹拖进去即可

### 🥈 Vercel
- 网址: https://vercel.com/
- 优点: 速度快，自动CI/CD
- 操作: 导入 GitHub 仓库或直接上传

### 🥉 GitHub Pages
- 网址: https://pages.github.com/
- 优点: 永久免费，稳定
- 操作: 推送到 GitHub 仓库，启用 Pages

## 💡 使用技巧

- **离线使用**: 第一次访问后自动缓存，之后无需网络
- **数据安全**: 所有数据保存在浏览器本地，不会丢失
- **自动更新**: 更新文件后，用户刷新页面即可获得新版本
- **卸载应用**: 长按图标 → 卸载（和普通 APP 一样）

## 🎨 自定义

### 修改图标
替换 `icon-192.png` 和 `icon-512.png` 为你的图标

### 修改应用信息
编辑 `manifest.json`：
```json
{
  "name": "你的应用名",
  "theme_color": "#你的颜色"
}
```

### 修改功能
所有代码都在 `index.html` 中，直接编辑即可

## 🐛 常见问题

**Q: 为什么没有安装提示？**
A: PWA 需要 HTTPS。使用 Netlify 等服务自动提供 HTTPS。

**Q: 能离线用吗？**
A: 可以！第一次访问后就能离线使用。

**Q: iPhone 上能用吗？**
A: 可以！手动添加到主屏幕即可。

**Q: 数据会丢失吗？**
A: 不会，除非清除浏览器数据。
