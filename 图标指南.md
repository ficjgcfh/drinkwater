# 应用图标设置指南

## 图标尺寸说明

Android 应用需要不同尺寸的图标以适配不同分辨率的设备。

### 所需图标尺寸

| 密度 | 尺寸 (px) | 文件名 | 用途 |
|------|----------|--------|------|
| ldpi | 36 x 36 | icon-ldpi.png | 低密度屏幕 |
| mdpi | 48 x 48 | icon-mdpi.png | 中密度屏幕 |
| hdpi | 72 x 72 | icon-hdpi.png | 高密度屏幕 |
| xhdpi | 96 x 96 | icon-xhdpi.png | 超高密度屏幕 |
| xxhdpi | 144 x 144 | icon-xxhdpi.png | 超超高密度屏幕 |
| xxxhdpi | 192 x 192 | icon-xxxhdpi.png | 超超超高密度屏幕 |

## 图标设计建议

### 设计规范
- 使用简单、清晰的图标
- 避免过多细节（小尺寸时难以辨认）
- 使用水滴💧主题符合应用定位
- 考虑在各种背景下的可见性

### 颜色建议
- 主色：蓝色/青色系（水的颜色）
- 可使用渐变效果
- 确保在深色和浅色背景下都清晰可见

### 图标形状
- Android 会自动应用遮罩层
- 建议使用圆形或圆角矩形设计
- 重要内容保持在安全区域内（距离边缘 10%）

## 创建图标的方法

### 方法一：使用在线工具

#### 1. App Icon Generator
网址: https://appicon.co/

步骤:
1. 上传一个 1024x1024 的原始图标
2. 选择 Android 平台
3. 下载生成的图标包
4. 将图标复制到 `www/img/` 目录

#### 2. Android Asset Studio
网址: https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html

步骤:
1. 上传图标或选择预设图标
2. 调整颜色、形状、填充
3. 下载生成的资源
4. 复制到项目目录

### 方法二：使用图像编辑软件

#### Photoshop / GIMP
1. 创建 512x512 的画布
2. 设计图标
3. 批量导出不同尺寸:
   - 保存为 PNG 格式
   - 透明背景
   - 使用图像缩放功能

#### Figma / Sketch
1. 创建图标设计
2. 使用插件批量导出不同尺寸
3. 导出为 PNG

### 方法三：使用 Cordova 插件

安装图标生成插件:
```bash
npm install -g cordova-res
```

准备一个 1024x1024 的图标文件 `icon.png`，放在项目根目录

运行:
```bash
cordova-res android --skip-config --copy
```

## 自动化脚本

创建 `generate-icons.sh`:

```bash
#!/bin/bash

# 需要安装 ImageMagick: brew install imagemagick

SOURCE="icon-source.png"  # 原始图标 (1024x1024)

mkdir -p www/img

convert $SOURCE -resize 36x36 www/img/icon-ldpi.png
convert $SOURCE -resize 48x48 www/img/icon-mdpi.png
convert $SOURCE -resize 72x72 www/img/icon-hdpi.png
convert $SOURCE -resize 96x96 www/img/icon-xhdpi.png
convert $SOURCE -resize 144x144 www/img/icon-xxhdpi.png
convert $SOURCE -resize 192x192 www/img/icon-xxxhdpi.png

echo "✅ 图标生成完成！"
```

使用方法:
```bash
chmod +x generate-icons.sh
./generate-icons.sh
```

## 配置文件

在 `config.xml` 中已经配置好了图标路径:

```xml
<platform name="android">
    <icon density="ldpi" src="www/img/icon-ldpi.png" />
    <icon density="mdpi" src="www/img/icon-mdpi.png" />
    <icon density="hdpi" src="www/img/icon-hdpi.png" />

</platform>
```

## 常见问题

### Q: 为什么构建时提示找不到图标?
A: 确保图标文件存在于 `www/img/` 目录，文件名正确。

### Q: 图标显示模糊怎么办?
A: 使用高质量的原始图标，确保是 PNG 格式，避免过度压缩。

### Q: 可以使用 JPG 格式吗?
A: 不建议。PNG 支持透明背景，显示效果更好。

### Q: 需要圆角吗?
A: 不需要手动添加圆角，Android 系统会自动处理。
