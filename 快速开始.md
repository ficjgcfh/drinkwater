# 🚀 快速开始 - 30秒体验应用

## 方法一：浏览器测试（最快！）

### Windows 系统
1. 双击 `test.html` 文件
2. 或右键 → 打开方式 → Chrome/Edge

### Mac/Linux 系统
```bash
open test.html
```

### 任何系统
```bash
cd drinkwater
./start.sh
# 选择选项 1
```

## 方法二：使用 Cordova 服务器

```bash
cd drinkwater
cordova serve
```

然后访问: http://localhost:8000

## 🎮 功能演示

### 1. 打卡测试
1. 点击中间的"打卡喝水"按钮
2. 观察：
   - 今日打卡次数 +1
   - 总计打卡次数 +1
   - 进度环前进
   - 记录出现在列表顶部
   - 显示成功提示

### 2. 多次打卡
1. 连续点击 8 次
2. 观察：
   - 进度环变满
   - 提示"已完成今日目标"

### 3. 清空记录
1. 点击右上角"清空记录"
2. 确认操作
3. 观察所有数据清空

### 4. 数据持久化测试
1. 打卡几次
2. 刷新页面（F5）
3. 观察数据依然保存

## 📱 Android 设备测试

### 需要 Android SDK 的方式

```bash
# 1. 检查环境
cordova requirements

# 2. 构建
cordova build android

# 3. 运行
cordova run android
```

### 不需要 Android SDK 的方式

1. 在有 Android SDK 的电脑上构建
2. 复制 APK 文件到手机
3. 直接安装

APK 位置:
```
platforms/android/app/build/outputs/apk/debug/app-debug.apk
```

## 💡 提示

- **第一次测试**: 使用浏览器版本 (`test.html`)
